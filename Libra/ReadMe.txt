Libra

-----------key board-------------
P/ space	enter the play mode/ respawn player ship   
esc		quit play mode/ quit the game
WASD		move around
IJKL		rotate the turret
C		change weapon
space     	shooting

`		Dev Console
F1		Debug mode
F4		Debug camera
F6		Heatmap mode
F8		restart the game
F9  		change to the next map

-----------controller------------
Start		enter the play mode/ respawn player ship     
Left joystick 	move around
Right joystick  rotate the turret
RT		shooting
X		change weapon
